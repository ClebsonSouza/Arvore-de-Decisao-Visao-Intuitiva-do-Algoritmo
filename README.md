Este projeto apresenta, de forma visual e didática, o funcionamento do algoritmo de Árvore de Decisão, um modelo de aprendizado de máquina que realiza previsões a partir de uma sequência de perguntas e divisões nos dados.

A lógica do algoritmo consiste em encontrar cortes que reduzam o erro e tornem os grupos mais homogêneos, permitindo classificar observações ou estimar valores numéricos.

Além da ideia central do modelo, o material também destaca conceitos importantes como ajuste do modelo, minimização do erro e overfitting, mostrando que árvores muito complexas podem se ajustar excessivamente aos dados de treino e perder capacidade de generalização em novos dados. 

O objetivo é oferecer uma explicação intuitiva, e útil para estudos, apresentações e introdução ao tema em ciência de dados.
